Exercices de révision

Exercice 0.

  • Problème 0 : Définissez une fonction Python print_even telle que, pour tout entier non négatif n, print_even(n) affiche tous les nombres pairs compris entre 0 et n.

  • Problème 1 : Écrivez deux fonctions, sum_rec et sum_iter, telles que, étant donné une liste l d'entiers, sum_rec(l) et sum_iter(l) calculent la somme (de manière itérative et récursive, respectivement) de tous les éléments de l.

  • Problème 2 : Écrivez une fonction search telle que, étant donné une liste l et une valeur e, search(l,e) renvoie True si e apparaît comme élément de l et False dans le cas contraire.

  • Problème 3 : Écrivez une fonction, occ, telle que, étant donné une liste l et une valeur e, occ(l,e) renvoie le nombre d'occurrences de e dans la liste l.

  • Problème 4 : Écrivez une fonction pairs telle que, étant donné une liste l, pairs(l) renvoie une liste de tous les tuples possibles (a,b) avec a et b dans l. Par exemple, pairs([1,2]) donne [(1,1),(1,2),(2,1),(2,2)].

  • Problème 5 : Écrivez une fonction count telle que, étant donné une liste l, count(l) liste toutes les paires (v,n) où v est une valeur de l et n est son nombre d'occurrences dans l. Par exemple, count([1,1,2,1,2,3]) donne [(1,3),(2,2),(3,1)].

  • Problème 6: Écrivez une fonction combis(l) qui, à partir d'une liste l, renvoie toutes les combinaisons sans répétition d'éléments de l. Par exemple, combis([1,2,3]) renvoie [(1,2), (1,3), (2,3)].

  • Problème 7: Écrivez une fonction ncombis(l,n) qui, à partir d'une liste l, renvoie toutes les n-combinaisons sans répétition d'éléments de l. Par exemple, combis([1,2,3,4],3) renvoie [(1,2,3), (1,2,4), (1,3,4), (2,3,4)].

  • Problème 8: Écrivez une fonction combisavec(l) qui, à partir d'une liste l, renvoie toutes les combinaisons avec répétition d'éléments de l. Par exemple, combis([1,2,3]) renvoie [(1, 1), (1, 2), (1, 3), (2, 2), (2, 3), (3, 3)].

  • Problème 9: Écrivez une fonction ncombisavec(l,n) qui, à partir d'une liste l, renvoie toutes les n-combinaisons avec répétition d'éléments de l. Par exemple, combis([1,2,3],3) renvoie [(1, 1, 1), (1, 1, 2), (1, 1, 3), (1, 2, 2), (1, 2, 3), (1, 3, 3), (2, 2, 2), (2, 2, 3), (2, 3, 3), (3, 3, 3)].

  • Problème 10: Écrivez une fonction bins2() pour générer toutes les chaînes binaires de taille 2. Autrement dit, bins2() doit renvoyer ["00","01","10","11"]. Essayez de le faire sans utiliser les fonctions précédemment implémentées.

  • Problème 11: Écrivez une fonction bins(n) pour générer toutes les chaînes binaires de taille n. Par exemple bins(3) doit renvoyer ["000","001","010","011","100","101","11","111"].

Exercice 1.

Le jeu de vie de Conway se déroule sur une grille, pour simplifier nous utiliserons une grille de 100 x 100, où chaque cellule est soit vivante, soit morte. À chaque étape, toutes les cellules sont mises à jour selon les règles suivantes:

  1. Une cellule morte possédant exactement trois cellules voisines vivantes devient vivante.

  2. Une cellule vivante ne possédant pas exactement deux ou trois cellules voisines vivantes meurt.

  3. Problème 0 : Déterminez comment modéliser les configurations c du jeu à l'aide des primitives Python que nous avons vues jusqu'à présent.

  4. Problème 1 : Définissez une fonction update(c) qui prend en entrée une configuration c et applique les règles ci-dessus à chaque cellule.

  5. Problème 3 : Définissez une fonction d'impression prettyprint(c) qui imprime une configuration c sur la sortie standard.

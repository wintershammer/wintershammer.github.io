Méthodologie de la programmation, TP 2.

Conseil général : consultez régulièrement la documentation Python.

Je suppose que vous avez mis en place un environnement approprié pour organiser votre travail (dossier dédié, éditeur de texte bien configuré).

Exercice 0.

  1. Créez un fichier .py et tapez-y python print("Hello world!").
  2. Utilisez le programme python3 pour interpréter le code source.

Exercice 1.

  1. Écrivez un programme qui affiche tous les nombres entiers de 0 à 20.
  2. Modifiez le programme ci-dessus afin que, pour chaque nombre entier \(i,\) il affiche Fizz si \(i\) est divisible par 3, Buzz s'il est divisible par 5.

Exercice 2.

  1. Écrivez une fonction isEven(i) qui, étant donné un entier i, renvoie python True si i est pair et python False sinon.
  2. Écrivez un programme qui affiche tous les nombres pairs entre 0 et 100.
  3. Écrivez un programme qui affiche la somme et le produit de tous les nombres pairs entre 0 et 100.
  4. Effectuez une modification minimale à votre programme afin qu'il effectue les tâches ci-dessus pour les nombres impairs.

Exercice 3.

  1. Écrivez une fonction area(i) qui prend en paramètre un nombre flottant r (rayon) et qui renvoie l'aire d'un disk de rayon r.
  2. Modifiez la fonction afin qu'elle prenne deux arguments : r,h, h étant par défaut égal à zéro. Si h est égal à zéro, la fonction doit renvoyer l'aire du disque comme auparavant. Si h est un nombre flottant positif, elle doit renvoyer le volume d'un cylindre de rayon \(r\) et de hauteur h.

Exercice 4.

  1. Écrivez une fonction qui prend trois entiers et renvoie leur moyenne arithmetique.
  2. Modifiez la fonction afin qu'elle prenne un argument type (par défaut arithmétique) : pour la valeur par défaut, la fonction renvoie comme ci-dessus la moyenne arithmétique. Si, au contraire, le type est géométrique, elle doit renvoyer la moyenne géométrique. Enfin, si la valeur est toute autre, elle doit afficher un message d'erreur et renvoyer None.

Exercice 5.

  1. Quelqu'un a déposé 100 euros sur son Livret A le 01/01/2025. Écrivez une fonction gains(n) qui prend en entrée un entier non négatif n et calcule le montant des intérêts accumulés sur n années. Pour ce faire, supposez que le taux d'intérêt annuel est fixé au taux actuel de 1,7 % et n'évolue pas.
  2. Supposons que le taux d'intérêt annuel lui-même évolue à un taux de 10 %. Modifiez votre fonction afin qu'elle effectue le calcul sur la base de cette nouvelle hypothèse.

Exercice 6.

  1. Un nombre \(n\) est carré s'il est égal à \(k^2\) pour un entier \(k\). Écrivez une fonction isSquare(n) qui prend en entrée un entier n et vérifie si il est carré, en renvoyant une valeur booléenne appropriée.
  2. Essayez de rendre votre programme efficace.

Exercice 7.

  1. Écrivez une fonction truthTable(f) qui prend en entrée une fonction booléenne unaire et affiche sa table de vérité : la valeur de f pour chaque entrée possible.

  2. Modifiez votre fonction afin qu'elle prenne en entrée une fonction booléenne binaire à la place.

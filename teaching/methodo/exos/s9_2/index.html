TP noté 2 - Modélisation SIR

Dans cet exercice, vous allez créer un petit simulateur de propagation d’épidémie basé sur le modèle SIR.

Nous utiliserons une version discrète en temps et non spatiale (modèle de population globale).

Le projet est organisé en plusieurs modules simples :

  • population.py : contient la classe Population
  • model.py : contient la classe SIRModel
  • simulation.py : contient la classe Simulation
  • display.py : contient les fonctions d’affichage

Rappel : modèle SIR

Une population est divisée en trois catégories :

  • S : susceptibles
  • I : infectés
  • R : retirés (guéris ou immunisés)

À chaque étape : - des individus passent de S → I avec un certain taux d’infection - des individus passent de I → R avec un certain taux de guérison

Exercice 0 — Classe Population

Implémentez dans un fichier population.py une classe Population qui représente l’état global de la population.

Spécifications :

  1. Le constructeur __init__ prend trois entiers S, I, R.

  2. La classe possède les attributs :

  3. S : nombre de susceptibles
  4. I : nombre d’infectés
  5. R : nombre de retirés

  6. N : population totale (fixe)

  7. Implémentez une méthode step(dI, dR) qui :

  8. enlève dI individus de S
  9. ajoute dI puis enlève dR individus à I

  10. ajoute dR individus à R

  11. Implémentez une méthode fractions() qui retourne les proportions : (S/N, I/N, R/N).

Exercice 1 — Modèle épidémique

Dans un fichier model.py, implémentez une classe SIRModel.

Cette classe représente la dynamique de l’épidémie.

Spécifications :

  1. Le constructeur prend deux paramètres :

    1. beta (taux d’infection)
    2. gamma (taux de guérison)

  2. Il y a deux méthodes :

    1. new_infections(pop) : retourne un entier correspondant au nombre de nouvelles infections dans la population pop.
    2. new_recoveries(pop) : retourne un entier correspondant au nombre de nouvelles guérisons.

On utilisera :

  • une loi binomiale avec paramètre pop.S et probabilité
    beta * pop.I / pop.N pour les infections,
  • une loi binomiale avec paramètre pop.I et probabilité gamma pour les guérisons,

et le morceau de code suivant pour simuler la distribution binomiale avec le paramètre p: 

import random

def binomial(n, p):
    """
    Sample from Binomial(n, p).
    """
    count = 0
    for _ in range(n):
        if random.random() < p:
            count += 1
    return count

Exercice 2 — Simulation

Dans le fichier simulation.py, implémentez une classe Simulation.

Cette classe gère l’évolution du modèle dans le temps.

Spécifications :

  1. Le constructeur prend :

    1. un objet Population
    2. un objet SIRModel

  2. La classe contient :

    1. un attribut t pour le temps courant
    2. un attribut (un dictionnaire, par exemple) history qui stocke l'évolution de S, I, R.

  3. Il y a deux methodes :

    1. une méthode step() qui effectue une itération :
      1. calcule les nouvelles infections
      2. calcule les nouvelles guérisons
      3. met à jour la population
      4. stocke les données dans history
    2. une méthode run(T) qui lance T itérations.

Exercice 3 — Expériences numériques

  1. Fixez N = 1000, I0 = 10, S0 = 990, R0 = 0.
  2. Testez différentes valeurs de beta et gamma.
  3. Que remarquez-vous sur :
  4. la hauteur du pic d’infectés ?
  5. la durée de l’épidémie ?
  6. la proportion finale d’individus infectés ?
